# 宠物美术验收清单（Codex 返工交付后逐项核）

> 配套返工规格：`docs/prompt-codex-pet-art-v2.md`。本清单 = 自动闸门 + 人工目检，两关都过才算达标。

## A. 自动闸门（必须先过）
```bash
python3 tools/check_pet_art.py     # 退出码 0 = 全过
```
逐项自动核验（阈值对齐 v2 §0）：
1. **文件齐全**：12 物种 × (var01-05 × A-D 立绘 + 8 动作条 + dex color/silhouette[/mystery])。
2. **尺寸**：立绘 512²；动作条 **4096×512（8 帧×512²）**；dex 512²。
3. **无截断**：任何一帧、任何边缘无实心内容触框（历次"头部被切"根治）。
4. **安全边距**：主体四周 ≥8% 透明边距。
5. **占比统一**：主体长边占画布 68–86%（v2 目标 78–82%）。
6. **动作帧稳定**：8 帧间主体尺寸波动 ≤12%（防脉动）、贴地基线波动 ≤6%（防上下跳）。
7. **剪影**：顶部不得被削平（防"头一刀切"），主体完整。

> 当前基线（返工前）：568 FAIL（多为占比/剪影/尺寸）。Codex 交付后应趋近 0。

## B. 人工目检（脚本查不了的，看接触表 `assets/art/qa/redo_<sp>_contact.png`）
逐物种确认：
- [ ] **画风**：奶油卡通水彩、成品渲染、可爱到位；无扁平 SVG/塑料/脏笔触感。
- [ ] **角色一致**：同物种跨阶段/动作/图鉴是同一只（配色/花纹/脸型一致）；关键辨识特征在（垂耳/龟壳/独角/盘蛇/鼓腮…）。
- [ ] **成长感**：A 幼(大头)→D 旅行(带旅装)，靠头身比+配件区分，占比恒定不忽大忽小。
- [ ] **动画流畅**：8 帧连续、平滑循环(8→1 不跳)、动机清晰(eat 有食盆/pat 有手+心/bath 有浴盆…)、不生硬。
- [ ] **剪影可辨**：纯色剪影一眼认得出是哪种动物，不糊成团、不缺头。
- [ ] **配件不越界**：围巾/背包/球/浴盆等都在安全边距内、不被切。
