# App Review 备注（简体中文）

Petopia 是一款完全离线、本地存档的单人陪伴游戏。

## 审核要点

- 无登录、无账号、无演示账号要求。
- 无广告、无应用内购买、无订阅。
- 无用户生成内容、聊天或联网社交功能。
- 游戏进度、宠物、图鉴与明信片均保存在设备本地。
- 通知默认关闭；只有用户在“设置 > 通知”主动开启时才请求系统权限。
- 通知只对应游戏内明信片、回访和纪念日，不发送通用召回提醒。
- “设置 > 存档与隐私”可通过系统分享面板导出存档，也可从用户明确选择的文件恢复。
- `ITSAppUsesNonExemptEncryption` 为 `false`；应用不实现专有加密。

## 建议审核路径

1. 首次启动浏览三页迎接画面。
2. 选择任一初始可领养物种并取名。
3. 在院子底部体验喂食、摸头、玩具与洗澡动作。
4. 点击顶部宠物信息卡查看宠物详情与成长手账。
5. 从右上角手账菜单进入图鉴、相册、商店、来客图鉴和设置。
6. 在设置中检查独立音乐/音效开关、通知分类、隐私说明及存档导入导出。

本地时间驱动的旅行、明信片、回访和每日来客会随游戏进度自然出现，不需要联网。
