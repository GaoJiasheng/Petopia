## 二、残留优化项（v9，可直接喂 codex）

> 统一原则：保持现有扁平水彩风；**浅色主体（兔/独角兽/淡色徽章）在透明/奶油底上对比不足 → 统一加一层柔和投影或加深主体轮廓**；每个图标 96px 一眼可辨。

### A. UI 图标 `ui_icon_*`（6 项）
| 文件 | 问题 | 改法 |
|---|---|---|
| `ach_firstadopt` | 粉点读不出"新生" | 改画**破壳雏鸟/裂开的蛋壳** |
| `ach_hidden_q` | 虚点线读不出"?" | 改画**清晰的水彩问号渍**（对齐图鉴 mystery 的问号） |
| `ach_visitor` | 与 `ach_yard` 都是小房子、太像 | 改画**脚印 + 小鸟**，与 yard(房子) 区分 |
| `hourglass_wc` | 虚点线读不出沙漏 | 改画**清晰水彩沙漏**（参照 `ui_btn_toggle_wc` 里那个沙漏，两者统一） |
| `flip` | 双色圆环偏抽象 | 改画**翻页/双箭头循环** |
| `shop_toy` / `shop_deco` | 偏糊 | toy=**毛线球/逗猫棒**、deco=**盆栽/小摆件**，主体更清晰 |

### B. UI 徽章 `ui_badge_*`（3 项）
| 文件 | 问题 | 改法 |
|---|---|---|
| `young` / `new_soft` / `revisit_pat` | 太淡、低对比，几乎看不见 | **提高饱和 + 加主体轮廓/投影**，确保在浅底可辨 |
| `graduate` vs `departure_sticker` | 都是蓝瓶子、太像 | `graduate` 改画**毕业帽/小背包**，与"离别贴纸"区分 |
| `raising_now` | 食盆读不出"在养中" | 改画**爱心 + 宠物剪影**（或小窝），表达"当前陪伴中" |

### C. 明信片贴纸 `pc_sticker_*`（~8 项，虚雾/抽象的那批）
仍偏"虚雾色团"、主体不明的：`blue_coat`（蓝宝石→改**蓝色小外套/斗篷**）、`smoke_ladders`、`spirit_letters`、`steam_hug`、`spring_stone`、`frost_fence`、`sweet_snow`、`scale_stars`、`sync_windmill`。
→ 提高主体清晰度/加深轮廓；氛围类可保留柔雾但需有一个可辨的核心物件。

### D. 明信片姿态 `pc_pose_*`（浅色物种）
`rabbit_*`、`uni_*`（兔/独角兽）主体过淡，在浅底几乎糊掉 → **加深轮廓或加柔和落影**，保证宠物主体清楚。帽子/相机道具本身已 OK。
