# Petopia 随机事件库（Content Pack）

> **Petopia 内容包 v0.3 · 配套 DESIGN.md · 本文件是【随机事件库】的权威来源**
>
> 更新日期：2026-07-02 ｜ 作者：内容组
>
> - 数据结构沿用 DESIGN.md §9.1 的 `Event` 定义；ID 规律 `ev_d##`（日常）/ `ev_s##`（特殊）。
> - 经验区间硬约束：**日常 +2~8 / 特殊 +8~20**；所有经验写入 `ExpLogEntry`（source = `EVENT_DAILY` / `EVENT_SPECIAL`，sourceRef = event.id）。
> - 暖绒奖励仅部分特殊事件携带，区间 **+10~30**。
> - 权重列写法：`性格×系数`（接 §2.3）、`天气/时段/季节×系数`、`需访客在场 / 需装饰 / 需豪华度` 等条件。
> - `限某物种` 表示该事件仅对该物种入池（实现建议：`weights.species` 字段 `[待细化]`，结构上等价于 requiresDecor 类条件）。
> - 带 **（分支）** 的事件含二选一 `choices`，分支各有专属短文与 `expDelta`（写法参照 ev_d12）；表中 exp 为基础值，`expDelta` 在其上 ±1~2。
> - 调性红线：手账体、1–3 句、笨拙可爱、有画面感；**零焦虑、绝不惩罚**——分支没有错误选项，两边都是好结局。
