# Petopia 宠物美术 · 完全返工规格 v2（Codex）

> 目标：一线手游品质的宠物立绘 + 逐动作动画。**统一、完整、饱满、可爱、流畅**。
> 现状问题（必须全部消除）：主体框选不一致、头/耳/尾被画框切、动作序列帧生硬/串帧、剪影糊成一团。
> 组织方式：**全局硬标准（§0）× 12 物种设定（§1）× 4 成长阶段（§2）× 8 动作（§3）**，三轴叠加定义每一张图。

---

## §0 全局硬标准（每一张图都必须满足，不可协商）

### 0.1 画风
- 奶油卡通水彩、圆钝可爱、成品渲染（对齐 `assets/art/samples/` Golden Set 与现有 cat/ember 的完成度）。
- 柔和描边或无描边、通透水彩上色、细节到位（毛发/鳞片/羽毛质感）。**禁止**扁平 SVG 感、线稿、AI 塑料感、脏笔触。
- 透明背景 PNG，无文字/水印/边框/相框。

### 0.2 构图与安全边距（这是"截断"问题的根治条款）
- 每帧画布 **512×512**，透明底。
- **主体必须完整**：头、耳、尾、四肢、道具**全部在画面内**，**任何一帧、任何部位都不得触碰或超出画布边缘**。
- **安全边距**：主体四周留 **≥10%（≥51px）透明边距**（顶部尤其严格——耳朵/独角/触角上方必须有空）。
- **主体占比统一**：主体**长边占画布 78–82%**（除天生细长的 snake 蜷成团后按此占比）。同一物种**跨阶段、跨动作、跨帧**占比一致，不得忽大忽小。
- **贴地基线统一**：主体**脚/底部落在画布 88–90% 处**（统一"站在地面"），所有帧一致，避免上下乱跳。
- **水平居中**：主体质心在画布水平中线附近；动作中的位移要小且回中，**不得把主体推到左右边缘**。

### 0.2b 静态↔动作 无缝一致（"播动作忽大忽小"的根治条款，极重要）
- 院子里平时显示的是**静态立绘**，点击/照料时**切到动作序列帧**。两者必须**无缝**：
  - 动作条(stageC)里主体的**屏上尺寸（占比）= 底图立绘的占比**（都 78–82%）；
  - 动作条主体的**贴地基线 = 底图基线**（都 88–90%）；
  - 动作**第 1 帧≈静态姿态**，从静态切入动作不跳变。
- 因动画在**任意成长阶段**都可能触发，故 **A/B/C/D 四档立绘与动作条的主体屏上尺寸必须彼此一致**（差异 <10%），否则一喂食宠物就"长大/缩小"，非常突兀。

### 0.3 动画质量（"动画太丑"的根治条款）
- 每个动作 = **横向序列帧条 `4096×512` = 8 帧 × 512²**（严格，所有物种一致）。
- **8 帧必须是同一只、同尺寸、同基线**的连续运动，**平滑循环**（第8帧接回第1帧不跳变）。
- 动作有**明确的动机弧线**（见 §3），不是抖动/缩放糊弄；关键帧之间补间自然。
- **绝不允许**：邻帧残片、主体在帧间大幅漂移、忽大忽小、糊成一团、帧内出现第二只。

### 0.4 一致性
- 同一物种在**所有阶段/动作/图鉴**里是**同一个角色**（配色、花纹、比例、脸型一致）。
- 变体 var01–05 = 同物种的**花色变种**（见 §1 每种的变体说明），造型骨架一致、只换配色/纹样。

## §2 四成长阶段（by 阶段）

> 同一角色的**成长**：越大越壮，比例从"大头幼态"到"匀称成体"。**占比恒定 78–82%**（不是靠占比表现大小，而是靠体态/比例/气质），阶段差异靠**头身比 + 姿态 + 配件**。

- **stage A｜幼**：大头短身（头身比≈1:1）、圆滚、姿态趴/蹲、眼睛占脸大、憨。最萌。
- **stage B｜少**：头身比≈1:1.4，开始有四肢线条，站姿或坐姿，活泼。
- **stage C｜成**：头身比≈1:1.8，匀称成体，标准坐/卧，最"标准"的一档（**动作动画基于此档**）。
- **stage D｜旅行/毕业**：成体 + **旅行配件**（围巾/小背包/鱼干挂饰等，各物种给一套合适的旅装），神态自信、准备远行。配件也必须在安全边距内、不被切。

> 每物种 × 每阶段 × 5 变体 = 240 张静态立绘 `pet_<sp>_var0N_stage[A-D].png`（512²，透明底，构图遵守 §0.2）。

## §3 八动作序列帧（by 动作）

> 均为 **stage C 成体**、**var01**、`4096×512` = 8 帧 × 512²、平滑循环、遵守 §0。动机弧线如下（8 帧描述关键运动）：

| 动作 file | 动机 | 8 帧运动弧线 |
|---|---|---|
| **idle** | 待机呼吸 | 身体极轻上下起伏 + 偶尔眨眼/耳动；1→8 平滑回到起点 |
| **eat** | 进食 | 面前有**食盆**；低头→叼食→咀嚼（腮动）→满足抬头；循环 |
| **pat** | 被摸头 | 头顶一只**温柔的手**轻抚，眯眼/歪头蹭手，冒 1–2 颗小爱心；手来回 |
| **play** | 玩耍 | 追/扑一个**小球或毛线**，前扑→拨弄→回弹；活泼但主体回中 |
| **bath** | 泡澡 | 坐在**小浴盆**里，泡泡浮动、拍水、舒服眯眼；水花小幅 |
| **sit** | 端坐 | 坐姿，尾巴/耳朵轻摆，左右张望；安静 |
| **sleep** | 睡觉 | 蜷起闭眼，均匀呼吸起伏，头顶飘一个小"Z"；极缓 |
| **walk** | 行走 | 四足/身体走一个循环步态（原地循环、位移≤帧宽5%，**不得走出画框**）；boo/snake 用飘移/滑行替代 |

> 物种适配：无手脚的 boo/snake/starbug 用身体动作替代（boo 飘晃、snake 盘身摆头吐信、starbug 星身脉冲发光）；parrot 立于栖木；cham 攀枝卷尾。道具（食盆/球/浴盆/手）风格统一、也在安全边距内。
