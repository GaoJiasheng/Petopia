# A. 12 种宠物完整设定档

> 通用说明：每物种含——基调 / 5 个模型变体（颜色花纹命名 + 一句差异描述）/ 4 成长档（A幼崽 Lv1–4 · B少年 Lv5–7 · C成年 Lv8–9 · D旅装 Lv10）外观演变要点 / 8 个通用基础动作（走/坐/睡/吃/被摸/洗澡/玩/待机）的物种化描述。
> 旅装（D 档）统一规则：成年体 + 小背包/围巾/旅行帽三选一主配件（按物种指定），细节见各档。

## A.1 橘猫 pet_cat

**基调**：慵懒、贪吃、晒太阳；院子里最理直气壮的存在，肚子永远比决心先到。

**模型变体（5）**：

| 变体 ID | 命名 | 差异描述 |
|---|---|---|
| cat_v1 | 全橘 | 从头橘到尾尖，晒太阳时像一块化开的橘子糖 |
| cat_v2 | 橘白 | 白手套白围兜，吃相再狼狈也显得斯文 |
| cat_v3 | 奶牛橘 | 橘白大块拼色，背上一块像地图的斑，据说指向饭碗 |
| cat_v4 | 浅杏 | 淡奶油杏色，虎纹若有似无，逆光时毛边泛金 |
| cat_v5 | 深炭橘 | 橘底压深棕虎纹，眼神看起来很酷，其实最馋 |

**成长档演变**：
- A 幼崽：头身比 1:1 的橘毛团，耳朵大得撑不起来，走路顺拐，肚皮蹭地。
- B 少年：身体拉长、腿变细，虎纹开始清晰，尾巴学会打问号，但跳跃仍常失手。
- C 成年：标志性的「橘胖」体态定型，卧姿如面包，眯眼时有稳稳的富态。
- D 旅装：脖子系一条格纹小围巾，背一个装小鱼干的迷你布包，包带被肚子撑得微微绷紧。

**通用基础动作（物种化）**：

| 动作 | 橘猫版描述 |
|---|---|
| 走 | 猫步中带着不情愿的拖沓，尾巴低垂画小圈，每三步蹭一下路边物 |
| 坐 | 前爪收拢的「面包坐」，肚子在地面摊出一圈软边 |
| 睡 | 侧躺→仰面→卷卷，一觉换三个姿势，肚皮随呼吸起伏最抢镜 |
| 吃 | 头埋进碗里只剩耳朵，进食声可视化为小音符，吃完舔碗沿 |
| 被摸 | 头往手心顶，眯眼、下巴抬高，喉咙咕噜声用水彩波纹表现 |
| 洗澡 | 全程「不情愿」名场面：耳朵向后压成飞机耳，湿身后瘦一圈的落差萌 |
| 玩 | 屁股高高摆动蓄力→扑空→装作只是想趴这儿 |
| 待机 | 蹲坐舔前爪洗脸，偶尔盯着空气中的浮尘出神 |

## A.2 柴犬 pet_shiba

**基调**：忠诚、傻乐、拆家未遂；笑容常驻，尾巴是永动的小螺旋桨。

**模型变体（5）**：

| 变体 ID | 命名 | 差异描述 |
|---|---|---|
| shiba_v1 | 赤柴 | 经典赤黄配白围脖，笑起来最标准的「柴式表情包」 |
| shiba_v2 | 黑柴 | 黑背棕眉点，两个「假眉毛」让每个表情都加倍生动 |
| shiba_v3 | 奶油柴 | 通体浅奶油色，像一团刚出炉的面包，脏得也最明显 |
| shiba_v4 | 胡麻柴 | 赤黑白三色芝麻纹混杂，跑起来毛色像被风搅动的落叶 |
| shiba_v5 | 焦糖斑 | 赤底带深焦糖色鞍状斑，尾巴尖一撮白像蘸了糖霜 |

**成长档演变**：
- A 幼崽：圆脸圆身滚地葫芦，耳朵还是软塌的三角形，跑两步绊自己一跤。
- B 少年：耳朵立起来了！腿长了、动作利落了，但刹车技术尚未同步升级。
- C 成年：胸膛挺起、卷尾饱满上翘，标志性咧嘴笑定型，坐姿端正如值班。
- D 旅装：头戴一顶小小的黄色旅行帽（帽檐有牙印），背登山款小背包，飞盘别在包侧。

**通用基础动作（物种化）**：

| 动作 | 柴犬版描述 |
|---|---|
| 走 | 小跑代替走，尾巴摇出残影，路线永远是波浪形 |
| 坐 | 端正军姿坐但尾巴出卖一切，扫地扫出一个小扇形 |
| 睡 | 四脚朝天大字型，舌头露一截，后腿偶尔梦中蹬两下 |
| 吃 | 三秒清盘，然后抬头无辜脸：「我吃了吗？」 |
| 被摸 | 整个身体往手上靠，摸到位时后腿不由自主跺地 |
| 洗澡 | 前半段享受后半段「柴犬拒否」式僵直，出浴抖水波及全院 |
| 玩 | 前肢趴地屁股撅起的「邀玩礼」，叼回玩具但不撒口 |
| 待机 | 坐着环视院子巡逻，看到什么都要「嗯？」地歪一下头 |

## A.3 垂耳兔 pet_rabbit

**基调**：软糯、胆小、爱啃；耳朵是两片安心的叶子，情绪都写在耳朵上。

**模型变体（5）**：

| 变体 ID | 命名 | 差异描述 |
|---|---|---|
| rabbit_v1 | 奶白 | 通体雪白粉鼻头，蹲着时像一颗大福 |
| rabbit_v2 | 灰蓝 | 烟灰蓝短绒，安静时几乎融进阴天的天色 |
| rabbit_v3 | 棕垂 | 榛果棕，耳朵内侧一圈浅奶色，像烤过头的边 |
| rabbit_v4 | 奶茶点点 | 奶茶底色撒可可斑点，背上斑点据说每只都不重样 |
| rabbit_v5 | 熊猫兔 | 黑白分明，眼圈自带黑眼罩，胆子却是全变体最小 |

**成长档演变**：
- A 幼崽：耳朵比身体还长，拖在地上走路会踩到，只敢在窝边一格活动。
- B 少年：会「binky」了（原地开心跳扭），耳朵能半抬表达情绪，活动圈扩大到半个院子。
- C 成年：绒毛蓬松如坐垫，站立远眺姿态解锁，耳朵情绪语言全套齐备。
- D 旅装：脖子系一个铃铛小丝巾（铃铛塞了棉花不响，因为怕吵到自己），背胡萝卜刺绣小挎包。

**通用基础动作（物种化）**：

| 动作 | 垂耳兔版描述 |
|---|---|
| 走 | 小步蹦跳，落地几乎无声，每五步停下确认四周 |
| 坐 | 前爪收在身下的「母鸡蹲」，耳朵自然下垂盖住半张脸 |
| 睡 | 侧翻「暴毙式」躺平（健康的信任表现），耳朵摊成两片叶子 |
| 吃 | 三瓣嘴高速咀嚼，胡萝卜以肉眼可见的速度缩短，腮边沾渣 |
| 被摸 | 顺耳朵方向摸会整只塌软，鼻子抽动频率下降表示满足 |
| 洗澡 | 只做「干洗」演出：湿毛巾轻擦 + 自己疯狂洗脸，全程耳朵警戒半竖 |
| 玩 | 推小球、钻纸筒，玩到兴头突然原地跳起转体（binky） |
| 待机 | 后腿站立张望两秒→落下→洗脸→再站立，循环警戒又好奇 |

## A.4 仓鼠 pet_hamster

**基调**：囤货、腮帮、跑轮；世界分两种东西——能塞进腮帮的，和暂时塞不进的。

**模型变体（5）**：

| 变体 ID | 命名 | 差异描述 |
|---|---|---|
| hamster_v1 | 金丝 | 金黄背毛白肚皮，经典款，腮帮鼓起时像两颗小金桔 |
| hamster_v2 | 布丁 | 通体奶黄，背上一道浅焦糖线，像布丁上的糖浆 |
| hamster_v3 | 银狐 | 雪白带银灰背线，冬天雪地里只能靠鼻子找到它 |
| hamster_v4 | 三线 | 灰底三条深背线，最小号变体，跑轮转速冠军 |
| hamster_v5 | 奶牛鼠 | 黑白块斑，囤货时黑白斑会随腮帮变形，喜感加倍 |

**成长档演变**：
- A 幼崽：花生米大小，腮帮还不会用，叼着一粒葵花籽跑来跑去不知放哪。
- B 少年：腮帮技能觉醒，第一次塞满时会被自己的脸吓到；跑轮从走到跑。
- C 成年：圆润梨形身材，囤货动线专业化（食盆→颊囊→窝底仓库三点一线）。
- D 旅装：背一个比自己还大的行军小包袱（其实很轻，都是种子壳），头顶迷你贝雷帽。

**通用基础动作（物种化）**：

| 动作 | 仓鼠版描述 |
|---|---|
| 走 | 贴地小碎步，尾巴几乎看不见，急停时会往前打滑半格 |
| 坐 | 直立坐，两只小手收在胸前，像随时准备鼓掌 |
| 睡 | 埋进垫材只露屁股，或在食盆里睡着（枕着粮食最安心） |
| 吃 | 双手捧食高速啃 + 塞颊囊二选一动画，腮帮渐进膨胀三档 |
| 被摸 | 用一根手指摸，它会抱住指尖闻一闻，然后继续忙自己的 |
| 洗澡 | 浴沙翻滚！进沙盆打三个滚，出来抖成一颗蓬蓬球 |
| 玩 | 跑轮狂奔（速度线特效），偶尔急停被惯性带着转半圈 |
| 待机 | 站立洗脸搓手，频率极高，像永远在准备开一个小会 |

## A.5 乌龟 pet_turtle

**基调**：慢悠悠、长情、缩壳；它的时间流速和别人不一样，但每一步都算数。

**模型变体（5）**：

| 变体 ID | 命名 | 差异描述 |
|---|---|---|
| turtle_v1 | 草绿 | 经典草龟绿，壳纹像一圈圈树木年轮 |
| turtle_v2 | 黄缘 | 深壳镶暖黄边，爬过草地时像移动的小灯笼边框 |
| turtle_v3 | 墨壳 | 近墨色壳面泛蓝光，雨天壳上水珠最好看 |
| turtle_v4 | 苔纹 | 绿底带深苔斑，趴着不动时常被访客误认成石头 |
| turtle_v5 | 蜜糖 | 浅蜜棕色，壳纹浅淡温柔，晒太阳时像一块烤年糕 |

**成长档演变**：
- A 幼崽：硬币大小，壳还软软的偏圆，翻倒了要蹬很久（你可以帮它）。
- B 少年：壳纹路清晰起来，学会自己翻身的「甩头绝技」，爬行速度提升 5%（它很自豪）。
- C 成年：壳面饱满有包浆光泽，晒背姿势稳如雕塑，脖子伸得更长更从容。
- D 旅装：壳上绑一个卷起的小铺盖 + 一顶斗笠挂在壳后，标准「行脚僧」造型。

**通用基础动作（物种化）**：

| 动作 | 乌龟版描述 |
|---|---|
| 走 | 四肢交替的庄严慢步，配极缓的上下起伏，背景可以走过一只快进的仓鼠作对比 |
| 坐 | 四肢收拢趴定，脖子留在外面，与「睡」的区别只有眼睛 |
| 睡 | 头和四肢全部收进壳，壳微微起伏，气泡式的鼾声符号 |
| 吃 | 慢慢瞄准→出其不意地快速一口（全身最快的动作），咀嚼回归慢速 |
| 被摸 | 摸壳它会把脖子伸到最长眯眼享受，摸头则害羞地缩一半 |
| 洗澡 | 全物种唯一的洗澡爱好者：泡在小盆里划水，壳刷出泡泡也纹丝不动 |
| 玩 | 顶小球——用头一下一下推，球滚出一米算进球，自己点头庆祝 |
| 待机 | 晒背，脖子随太阳角度缓慢调整，像一台生物日晷 |

## A.6 鹦鹉 pet_parrot

| parrot_v2 | 蓝黄金刚(迷你) | 宝蓝翅金黄胸，圆珠笔大小的自信，走路自带王者步 |
| parrot_v3 | 灰红尾 | 烟灰身板红尾羽，学舌最像，音色自带电台质感 |

| 睡 | 头埋进背羽，呆毛（如有）仍倔强地立着 |
| 吃 | 用脚抓着食物送嘴边啃，吃瓜子壳精准弹开，一地狼藉 |
| 被摸 | 低头把后颈送过来，眼睛半闭，被挠对位置时冠羽缓缓竖起 |
| 洗澡 | 水碗扑腾式：跳进去疯狂抖翅，水花四溅，出来湿成一根「鸟签」 |
| 玩 | 拆铃铛玩具：咬、甩、拽，最后把铃铛叼到最高处扔下去听响 |
| 待机 | 自言自语练词，偶尔突然大声广播一句你上周说过的话 |

## A.7 玉米蛇 pet_snake

**基调**：高冷、蜷缩、蜕皮；话最少存在感最稳，爱你但表达方式是「在你附近盘着」。

**模型变体（5）**：

| 变体 ID | 命名 | 差异描述 |
|---|---|---|
| snake_v1 | 橙红经典 | 橙底红斑黑描边，盘起来像一圈暖色的年轮蛋糕 |
| snake_v2 | 奶油白化 | 奶白配浅橙斑、红宝石眼，晒太阳时半透明发光 |
| snake_v3 | 薰衣草 | 灰紫底粉斑，全变体最梦幻，月夜下泛淡紫 |
| snake_v4 | 炭灰 | 黑灰斑纹冷都款，与本体反差最大（其实最爱钻袖子） |
| snake_v5 | 糖果条纹 | 红白相间长条纹，被访客集体误认为拐杖糖 |

**成长档演变**：
- A 幼崽：铅笔长短细细一条，斑纹是浅浅的水彩点，爱躲在花盆边只探出头。
- B 少年：第一次蜕皮事件（旧皮完整保留可收藏进手账），身体变长变亮。
- C 成年：体色饱和斑纹立体，盘踞姿态优雅，会在固定的「宝座石」上定点值守。
- D 旅装：不能背包，于是把一条极细的围巾松松绕在颈后（它自己挑的），行李是一个由回访宠代为托运的小木箱。

**通用基础动作（物种化）**：

| 动作 | 玉米蛇版描述 |
|---|---|
