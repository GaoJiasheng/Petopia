# Petopia 🐾

一个手绘水彩（奶油可爱风）的小院子养宠手游。一次只养一只宠物，陪它从幼崽长大，满级后它「毕业」去环游世界——不是告别：它会持续寄回明信片，偶尔回院子串门。送走一只，再迎来下一只。院子里不时有野生动物访客路过，慢慢填满你的图鉴。

- **平台**：iOS + Android（Flutter + Flame）
- **调性**：治愈、零焦虑、绝不惩罚离线
- **状态**：设计完成，开发即将开始
